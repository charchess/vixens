## 1. Dictionnaire des Poses & Expressions (Core Poses)
Tout agent (plugin) doit fournir un mapping pour ces IDs. Un ID peut pointer vers un groupe d'images pour introduire de l'aléatoire.

| Emotion ID | Intention Visuelle | Dynamique (VFX) |
| :--- | :--- | :--- |
| `idle` / `neutral` | Posture stable, regard utilisateur. | Micro-respiration CSS. |
| `thinking` | Regard haut, main sur menton. | Floating dots (points flottants). |
| `happy` / `joy` | Sourire, posture ouverte. | Sparkles (étincelles dorées). |
| `sad` / `sorrow` | Épaules tombantes, regard bas. | Rain drops (pluie subtile). |
| `alert` | Sourcils froncés, corps tendu. | Glow pulse orange sur la bordure. |
| `emergency` | Posture grave, air sérieux. | Red screen alert (teinte rouge). |
| `confused` | Tête penchée, air perplexe. | Question mark filaire. |
| `shy` / `blush` | Regard de côté, légère rougeur. | Soft pink glow sur les joues. |
| `angry` | Regard noir, sourcils bas. | Screen shake (secousse UI). |
| `glitch` | Pose déformée, artéfacts. | Scanlines & Chromatic aberration. |

## 2. Charte Sonore & Synchro (Soundscape)
Les sons doivent rester organiques ("Cyber-Cozy").

### 2.1 Effets Sonores (SFX Mapping)
*   **Success:** `sfx_chime` - Carillon léger.
*   **Error (Glitch):** `sfx_static` - Bruit blanc / Bitcrush.
*   **Alert P1:** `sfx_double_beep` - Médium, ton neutre.
*   **Urgence P0:** `sfx_deep_drone` - Basse pulsée.
*   **Text Typing:** `sfx_typewriter` - Clic mécanique étouffé.
*   **Agent Movement:** `sfx_breeze` - Souffle d'air doux.

### 2.2 Ambiances (Loops)
*   `ambience_night`: Pluie douce + craquements de bois.
*   `ambience_work`: White noise technique + ventilo lointain.

## 3. Standards Techniques des Assets

### 3.1 Sprites (Personnages)
*   **Format:** WebP (Lossless) ou PNG-24 avec transparence.
*   **Résolution:** 2000px de hauteur minimum.
*   **Pivot:** Bottom-Center (0.5, 1.0).
*   **Lip-Sync (Optionnel):** Fournir `mouth_open` et `mouth_closed` pour animation par amplitude.

### 3.2 Icônes (Totems)
*   **Format:** SVG (Vectoriel).
*   **Style:** Minimalisme "Flat", pas de dégradés.
*   **Couleur:** Manipulable via CSS (fill: currentColor).

---

## 4. Logique de Synchronisation (A/V Sync)
*   **Pose Anticipation:** Le sprite change **200ms avant** le début du texte.
*   **Typewriter Sync:** La vitesse de texte est calée sur la durée du fichier audio TTS.
*   **Pulsation Lumineuse:** L'aura de l'Ardoise suit l'amplitude Web Audio en temps réel.

---

## 5. Immersion Avancée (Glitch Mode)
Lorsqu'un message est marqué comme `corrupted`:
1.  **Visuel:** Application du filtre Scanlines sur le Stage.
2.  **Texte:** Substitution de 5% des caractères par des symboles Unicode (`░`, `█`, etc.).
3.  **Audio:** Application d'un filtre Bitcrusher sur le flux TTS.
